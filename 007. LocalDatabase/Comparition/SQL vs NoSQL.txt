+------------------+--------------------------------------------+------------------------------------------------------+
|                  |                     SQL                    |                         NoSQL                        |
+------------------+--------------------------------------------+------------------------------------------------------+
| Cấu trúc dữ liệu | Lưu dạng bảng                              | Lưu dạng Key-Value                                   |
|                  | Các bảng có quan hệ với nhau               | Có thể tổ chức dạng cây Json                         |
+------------------+--------------------------------------------+------------------------------------------------------+
| Thiết kế         | Thiết kế từ ban đầu, có cấu trúc           | Linh hoạt, dễ thay đổi                               |
|                  | Thay đổi khó khăn                          |                                                      |
+------------------+--------------------------------------------+------------------------------------------------------+
| Chuẩn hóa        | Hổ trợ chuẩn hóa, hạn chế dư thừa          | Nếu chuẩn hóa thì công tác truy xuất rất tốn kém     |
|                  | Hỗ trợ Join bảng                           | Nếu không chuẩn hóa thì cập nhật rất tốn kém         |
+------------------+--------------------------------------------+------------------------------------------------------+
| Giao dịch        | Thực hiện đồng loạt trên nhiều record      | Thực hiện trên từng record                           |
+------------------+--------------------------------------------+------------------------------------------------------+
| Hiệu suất        | SQL có hỗ trợ Join để truy xuất nên hiệu suất chậm hơn NoSQL với tổ chức dữ liệu không chuẩn hóa, |
|                  | có thể lấy tất cả thông tin về một đối tượng, không cần xem xét các đối tượng liên quan.          |
+------------------+---------------------------------------------------------------------------------------------------+
